# PCM Recon

Desktop save parser and scouting tool for Pro Cycling Manager career files.

[![Install](https://img.shields.io/badge/Install-Windows%20Build-e8b800?style=for-the-badge)](https://github.com/Venutian/pcm-recon/releases/tag/v2.0.2)

[![Buy Me Coffee](https://img.shields.io/badge/Donate-Ko--fi-ff5f5f?style=for-the-badge)](https://ko-fi.com/venutian)

## Download

Get the latest Windows build from the [Releases page](https://github.com/Venutian/pcm-recon/releases/tag/v2.0.2).

## What It Does

- Opens PCM career `.cdb` saves
- Browses riders, teams, rankings, and free agents
- Shows dedicated scout report prospects and potentials
